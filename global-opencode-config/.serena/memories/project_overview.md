`~/.config/opencode` is the global OpenCode configuration workspace. It contains the main `opencode.json` config, AGENTS/instruction files, reusable skills, plugins, scripts, rules, and auxiliary docs for the local OpenCode environment. Top-level structure includes `skills/`, `instructions/`, `agents-instructions/`, `commands/`, `tools/`, `rules/`, `scripts/`, and config files like `opencode.json`, `mcp.json`, `AGENTS.md`, and `SKILLS.md`. The stack is primarily JSON/Markdown/TypeScript-flavored config and scripting.
