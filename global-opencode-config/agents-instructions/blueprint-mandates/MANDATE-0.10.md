# MANDATE 0.10: COMMIT MESSAGE STANDARDS

**Conventional Commits required.**

Format: `<type>(<scope>): <description>`

Types:
- `feat:` New feature
- `fix:` Bug fix
- `docs:` Documentation
- `style:` Formatting
- `refactor:` Code restructure
- `test:` Tests
- `chore:` Maintenance

Example: `feat(auth): implement Antigravity OAuth flow`
