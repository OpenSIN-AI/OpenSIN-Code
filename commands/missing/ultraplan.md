# /ultraplan

Ultra-planning with multi-pass architecture design.

## Usage

```bash
/ultraplan Add real-time collaboration feature
/ultraplan Migrate to microservices architecture
```

## Planning Phases

1. **Requirements Analysis** — What needs to be built
2. **Current State Analysis** — What exists now
3. **Gap Analysis** — What's missing
4. **Architecture Design** — How to build it
5. **Implementation Plan** — Step-by-step plan
6. **Risk Assessment** — What could go wrong
7. **Timeline Estimate** — How long it takes
8. **Resource Planning** — What's needed

## Output

Detailed planning document saved to `.opensin/plans/<feature-name>.md`
