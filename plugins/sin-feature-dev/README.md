# sin-feature-dev Plugin

7-Phase Feature Development mit Multi-Agent Workflow für OpenSIN-Code.

## Quick Start

```bash
/feature-dev Add user authentication with OAuth
```

## The 7 Phases
1. **Discovery** — Understand what needs to be built
2. **Codebase Exploration** — Launch code-explorer agents
3. **Clarifying Questions** — Fill in gaps
4. **Architecture Design** — Launch code-architect agents
5. **Implementation** — Build with explicit approval
6. **Quality Review** — Launch code-reviewer agents
7. **Summary** — Document accomplishments

## Agents
- `code-explorer` — Traces execution paths
- `code-architect` — Designs feature architectures
- `code-reviewer` — Reviews for bugs and quality
