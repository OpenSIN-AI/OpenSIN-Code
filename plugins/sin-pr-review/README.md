# sin-pr-review Plugin

6 spezialisierte Review-Agents für Code-Qualität in OpenSIN-Code.

## Agents

| Agent | Focus | Rating |
|-------|-------|--------|
| comment-analyzer | Comment accuracy | Accuracy 1-10 |
| pr-test-analyzer | Test coverage | Gap severity 1-10 |
| silent-failure-hunter | Error handling | Severity levels |
| type-design-analyzer | Type design | 4 dimensions × 1-10 |
| code-reviewer | General quality | Score 0-100 |
| code-simplifier | Simplification | Before/after |
