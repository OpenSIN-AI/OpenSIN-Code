# /hookify:configure

Interactively enable or disable existing hookify rules.

## Usage

```
/hookify:configure
```

## What this does

1. Lists all existing rules with their current status
2. Lets you toggle rules on/off by editing `enabled: true/false`
3. Changes take effect immediately
