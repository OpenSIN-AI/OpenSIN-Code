# sin-plugin-dev Toolkit

Plugin Development Toolkit für OpenSIN-Code — 7 Skills + 8-Phase Create-Plugin Workflow.

## Skills

1. **Hook Development** — Advanced hooks API
2. **MCP Integration** — MCP server integration
3. **Plugin Structure** — Plugin organization
4. **Plugin Settings** — Configuration patterns
5. **Command Development** — Slash commands
6. **Agent Development** — Autonomous agents
7. **Skill Development** — Skills with progressive disclosure

## Commands

- `/plugin-dev:create-plugin` — 8-phase guided plugin creation
- `/plugin-dev:help` — Help
